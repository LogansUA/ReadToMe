# ReadToMe
Read to me

### Installation
* Install [PhantomJS](http://phantomjs.org/)
* Install [Festival](http://festvox.org/)
```
sudo apt-get install festival festvox-kallpc16k
```
* Download ReadToMe:
```
git clone https://github.com/LogansUA/ReadToMe.git
```

### Usage
```
./pasp.sh <url> [jQuery selector]
```
